## What is Fashionista RTX Edition?

Fashionista RTX Edition is a Vampire the Masquerade: Bloodlines list that I put together so I could try out the RTX Mod. The mod is Built to use the Unofficial Patch as a Base and then overhaul all the graphics.

### Is the Mod complete

No.

The mod is currently in beta and should be treated as such.

### When is the list updated?

I will try to update the list whenever there is an update to the RTX mod.

### FAQ

- When you launch the game there should be a custom Menu and a message in the top corner of the screen informing you that RTX mod is running if you do not see this then reinstall the list or check the discord for support.
- The shortcut combo to open the RTX Menu is ``alt + x`` 
- This is designed for Nvidia Cards but if you have a recent AMD card it *might* work.
- This is a beta and there will be bugs accept that and DO NOT bother the mod author.
