# {{ page.title }}

**VTMB HD Overhaul** - is a comprehensive visual enhancement mod that updates Vampire: The Masquerade - Bloodlines with high-definition textures, improved character models, enhanced lighting effects, and upgraded environmental graphics to bring the classic game up to modern visual standards.

{: .warning}
> **New Game Required** - You must start a new game when switching to this profile. Do not use save files from other profiles or mod versions.

More information about VTMB HD Overhaul can be found on [Nexus](https://www.nexusmods.com/vampirebloodlines/mods/234).
