# {{ page.title }}

**Bloodlines Prelude I** is an episodic mod for Vampire: The Masquerade - Bloodlines that shows the events leading up to the story of the original game. This time you start as a human hunter from the Society of Leopold.

This is Part 1 of a planned 3, according to updates from the creator Part 3 is not expected for 3-5 years.

{: .warning}
> **New Game Required** - You must start a new game when switching to this profile. Do not use save files from other profiles or mod versions.

More information about Bloodlines Prelude I can be found on [ModDB](https://www.moddb.com/mods/vtmb-prelude/downloads/bloodlines-prelude-i-v18).
