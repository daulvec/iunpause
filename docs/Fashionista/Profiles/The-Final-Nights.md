# {{ page.title }}

**The Final Nights: Reborn** is a restoration project for this classic expansion, updated for the latest Unofficial Patch.

{: .warning}
> **New Game Required** - You must start a new game when switching to this profile. Do not use save files from other profiles or mod versions.

More information about The Final Nights: Reborn is available on [ModDB](https://www.moddb.com/mods/the-final-nights-reborn).

This expansion intentionally drops the "B" from VTMB to highlight that it offers a new experience rather than a direct continuation of Bloodlines. It includes 7 new clans, 5 new disciplines, extra quests, new NPCs, new game mechanics, and new music.

The profile builds on Camarilla Edition systems like the blood timer, pedestrian blood dolls, and updated combat.
