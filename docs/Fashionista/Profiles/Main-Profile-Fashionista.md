# {{ page.title }}

This is the main and recommended profile for the Fashionista modlist. Most of the testing and development work has focused on this profile, making it the most stable and feature-complete experience.

{: .warning}
> **New Game Required** - You must start a new game when switching to this profile. Do not use save files from other profiles or mod versions.

## Base Foundation

Built on the **Unofficial Patch Plus Edition**, which adds additional tweaks and lots of new content beyond the standard Unofficial Patch.

More information about the Unofficial Patch can be found on the [Nexus](https://www.nexusmods.com/vampirebloodlines/mods/80).

## Some of the more important mods to be aware of

### Visual Enhancements
- **HD Textures** - [High-resolution texture replacements throughout the game](https://www.nexusmods.com/vampirebloodlines/mods/34)
- **NPC Overhauls** - [Updated character models and appearances](https://www.nexusmods.com/vampirebloodlines/mods/165)

### Audio Improvements  
- **Remastered Voices** - [Enhanced voice quality and clarity](https://www.nexusmods.com/vampirebloodlines/mods/307)
- **Improved Music** - [Remastered soundtrack and ambient audio](https://www.nexusmods.com/vampirebloodlines/mods/324)

### Fashion & Outfits
- **Extensive Outfit Collection** - [Almost every outfit mod available for the game](/Fashionista/Fashion.html))
- **Curated Selection** - My personal favorite outfits enabled by default
