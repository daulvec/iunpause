# {{ page.title }}

**The Hecata Chronicles** - In this mod you will start during the event of Bloodlines but from a Giovanni point of view, it will go up to the Familly Reunion. At some point Harbringer will be added.

{: .warning}
> **New Game Required** - You must start a new game when switching to this profile. Do not use save files from other profiles or mod versions.

More information about The Hecata Chronicles can be found on [ModDB](https://www.moddb.com/mods/hecata-chronicles).
