# {{ page.title }}

**Bloodlines: Antitribu** is a comprehensive overhaul mod that enhances and expands the Vampire: The Masquerade - Bloodlines experience with new content, improved difficulty, and enhanced visuals.

{: .warning}
> **New Game Required** - You must start a new game when switching to this profile. Do not use save files from other profiles or mod versions.

Link to Bloodlines Extreme on [ModDB](https://www.moddb.com/mods/bloodlines-hard-difficulty/)

## Options Selected

- **Hunter pack**  In initial equipment, all Hunters 'll have the "Sword Of The Light" (from broadsword model) doing a good damage and with a nice look.
- Default Gold Color for the Anaconda
- Default Soundtrack options
- BE Defaults for the Taxi (Brighter Yellow Color)
- **Nosferatu pack** Makes Playing as them easier
  - More XP
  - They can Walk or Run in the Street if you stay far enough way from humans.
  - Killing Hunters is not an auto masquerade violation
  - Double Masquerade redeption 
  - More action, dialogue, quest and a Clan Specific Worldmap
- All the Skins for the Play
  - Brujah - Male 0-3
  - Gangrel - Male 3
  - Malkavian - Male 0-3
  - Nosferatu - Nake 0-3
  - Tremere - Male 0-3
  - Tremere - Female 0-3
  - Ventrue - Male 0-3
- The Textures Pack
