This section covers the various alternative profiles available in the modlist. Each profile offers a unique gameplay experience with different mods and features.

## Available Profiles

| Profile | Description |
| ------ | ----------- |
| [**Main Profile - Fashionista**](Profiles/Main-Profile-Fashionista.html) | **Recommended** - The most stable and feature-complete profile with HD textures, extensive outfit collection, and enhanced visuals. Built on Unofficial Patch Plus Edition. |
| [**Base Game + Unofficial Patch**](Profiles/Base-Game-Unofficial-Patch.html) | Minimal setup with just the base game and bug fixes. Perfect for vanilla experience with stability improvements. |
| [**Antitribu**](Profiles/Antitribu.html) | Enhanced difficulty overhaul with Hunter packs, improved Nosferatu gameplay, masquerade tweaks, and additional skins. |
| [**Barabbah Overhaul**](Profiles/Barabbah-Overhaul.html) | Comprehensive mod that enhances and expands the base game experience. |
| [**Bloodlines Extreme**](Profiles/Bloodlines-Extreme.html) | Hardcore overhaul with enhanced difficulty, Hunter equipment upgrades, and special Nosferatu features for challenging gameplay. |
| [**Bloodlines Prelude I**](Profiles/Bloodlines-Prelude-I.html) | First part of the prequel story expansion that adds new content before the main campaign. |
| [**Bloodlines Prelude II**](Profiles/Bloodlines-Prelude-II.html) | Continuation of the prequel story with additional quests and storylines. |
| [**Clan Quest Mod**](Profiles/Clan-Quest-Mod.html) | Major expansion adding clan-specific quests, Sabbat path, and integration of multiple popular mods including Arsenal and Companion Mod. |
| [**Companion Mod Core Edition**](Profiles/Companion-Mod-Core-Edition.html) | Adds recruitable companions and expanded dialogue options throughout the game. |
| [**Camarilla Edition Reborn**](Profiles/Camarilla-Edition-reborn.html) | Complete mechanics overhaul focusing on realistic vampire survival, feeding systems, and sandbox-style gameplay. |
| [**The Hecata Chronicles**](Profiles/The-Hecata-Chronicles.html) | Story expansion focusing on the Hecata clan with new quests and content. |
| [**La by Night**](Profiles/La-by-Night.html) | Themed experience based on the popular Vampire: The Masquerade actual play series. |
| [**VTMB HD Overhaul**](Profiles/VTMB-HD-Overhaul.html) | High-definition texture and model improvements for enhanced visual fidelity. |
| [**War Games**](Profiles/War-Games.html) | Combat-focused profile with enhanced action gameplay and tactical elements. |
